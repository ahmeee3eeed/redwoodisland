## 🎨 IMAGE OPTIMIZATION TIPS

Before adding images to your website:

1. **Resize images** to recommended dimensions
   - Use tools like:
     - **Photopea** (free, online Photoshop alternative)
     - **GIMP** (free desktop software)
     - **Canva** (easy online tool)

2. **Compress images** to reduce file size
   - Use: https://tinypng.com/ or https://compressor.io/
   - Target: Under 200KB per image
   - Benefits: Faster loading website

3. **Use correct formats**
   - **JPG** for photos (fruits, facilities)
   - **PNG** for logos (transparent backgrounds)
   - **WebP** for modern browsers (optional)

## 🔧 TROUBLESHOOTING

**Image not showing?**
1. Check file name matches exactly (case-sensitive!)
   - "Kiwi.jpg" ≠ "kiwi.jpg"
2. Check image is in correct folder
3. Check file extension (.jpg, .png, not .jpeg)
4. Refresh browser (Ctrl+F5 or Cmd+Shift+R)

**Image looks blurry?**
- Use higher resolution source image
- Don't stretch small images

**Page loads slowly?**
- Compress your images
- Aim for <200KB per image
