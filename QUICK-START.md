## 📝 QUICK CUSTOMIZATION

### Change Contact Info
**Find and replace in ALL HTML files:**
- **Phone:** +966-75-7557363 → Your phone
- **Email:** info@redwoodisland.com → Your email
- **Location:** Dubai, UAE → Your location

**How to find/replace:**
1. Open HTML file in Notepad/TextEdit
2. Press Ctrl+H (Windows) or Cmd+F (Mac)
3. Type old text, then new text
4. Click "Replace All"
5. Save file

### Change Colors
Open `css/style.css` and change at the top:
```css
--primary-green: #2d5f3f;    /* Your main color */
--accent-yellow: #c8d91d;    /* Your accent color */
```

## ⚠️ COMMON ISSUES & FIXES

### "Images not showing"
- ✅ Check file names match exactly
- ✅ Check images are in correct folders
- ✅ Refresh browser (Ctrl+F5)

### "Website looks broken on mobile"
- ✅ Make sure style.css is in css/ folder
- ✅ Check no files are missing
- ✅ Clear browser cache

### "Contact form doesn't work"
- ✅ Currently shows alert message
- ✅ To actually send emails, you need:
  - Form handling service (Formspree, EmailJS)
  - Or backend PHP script
  - Easy to add later!
